# Perspective.js

 Create 3D-like elements on scroll (no jQuery)
 
 Usage: new Perspective( [element], [amount] )
 
 Example:
```javascript
new Perspective('h1', 150) // Turn all h1 tag into perspective elements
new Perspective('img', 80) // Image will move less than h1
```
